# 목적

이 문서는 Dashboard를 제외한 모든 페이지의 UX와 UI를 정의한다.

모든 페이지는 동일한 Design System과 Layout System을 사용한다.

Sidebar와 Header는 모든 페이지에서 동일하게 유지한다.

사용자는 페이지를 이동해도 다른 서비스처럼 느껴져서는 안 된다.
